Scratch Project

Project Name: A Small Story
Course: Computer Science Priniciples (CSP)
Duration: About a week (September 3, 2019 - September 11, 2019)
Objective: To code either a pick your own adventure or some type of game in Scratch
Type of Project: Group Project

For this project, I was responsible for the code. My partner was responsible for the documentation, however, we both worked on the coding and the documentation. We were provided with a document that contained the criteria for both options and some questions for when we finished coding the game or story. While coding our project, (we chose to do a story) we found that all the timings had to be near exact otherwise everything that followed would be off. We also found out that it is better to use fewer sprites and broadcasts to prevent and/or lessen lag.
